\# 猫眼电影TOP100数据分析



这是一个基于猫眼电影TOP100榜单的数据分析项目，使用Python进行数据爬取和处理，通过网页展示数据分析结果。



\## 项目特点



\- 数据概览统计

\- 年代分布分析

\- 评分分布分析  

\- 地域分布分析

\- 演员影响力分析

\- TOP10电影排行榜

\- 完整数据浏览



\## 技术栈



\- 前端：HTML5 + CSS3 + JavaScript

\- 图表库：ECharts

\- UI框架：Bootstrap

\- 数据获取：Python + Requests + BeautifulSoup



\## 部署说明



本项目使用GitHub Pages进行部署，访问地址：https://\[你的用户名].github.io/movie-top100-analysis/



\## 数据来源



数据来源于猫眼电影TOP100榜单：https://maoyan.com/board/4



\## 使用说明



1\. 在data.js文件中替换为你的真实数据

2\. 将文件推送到GitHub仓库

3\. 在仓库设置中启用GitHub Pages

4\. 访问生成的URL查看网页
